Are you tired of having to clear a huge area full of plants just to build your houses, bases, or machines?
Now, with the Simple Machete, you can do it quickly using just your Sweeping-enchanted sword.

![image](https://github.com/user-attachments/assets/9b628bd9-f92e-4d21-bf58-3f8039309d89)
## Features
- **Bush cleaning**: By using a sword enchanted with Sweeping Edge, you can clear leaves, moss, grass, etc. within a specified radius.
- **WorldGuard Support**: **"sm-break"** flag allows Machete to use within a region or not 

![image](https://github.com/user-attachments/assets/31b13b0c-d8e2-4634-813e-280ae6d081d7)
## Commands
- `/sm reload` reloads the configuration file.
## Permissions
- `simplemachete.admin` Allows admin command.
- `simplemachete.use` Allows to use Machete.
