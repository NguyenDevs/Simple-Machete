## Simple Machete Plugin

**Tired of spending hours breaking every single leaf, vine, moss, and tall grass just to clear space for your build?**  

With **Simple Machete**, turn any **Sweeping Edge enchanted sword** into a powerful bush-clearing tool!  
One swing and watch huge areas of plants disappear instantly — perfect for building houses, mega-bases, redstone contraptions, or just cleaning up your survival world.

Fast. Satisfying. Survival-friendly.

## Features
- **Bush Clearing with Sweeping Edge**  
  Swing your Sweeping Edge enchanted sword → instantly clear leaves, vines, moss blocks, grass, tall grass, ferns, flowers, and more in a configurable radius.  
  Feels just like using a real machete!

- **WorldGuard Integration**  
  Use the custom flag **`sm-break`** to allow or deny machete clearing inside specific regions.  
  Perfect control for spawn areas, minigames, or protected zones.

- **Lightweight & Configurable**  
  Adjust radius, affected blocks, cooldown, durability usage, and more — all from a simple config file.

## Commands
- `/sm reload`  
  Reloads the configuration file (admin only)

## Permissions
- `simplemachete.admin`  
  Access to admin commands (`/sm reload`)

- `simplemachete.use`  
  Allows players to use the machete clearing feature with a Sweeping Edge sword
